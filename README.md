# 🔥Features

📫 Send all kinds of messages, files and images

🎙️ Voice messages

📍 Location sharing

🉑️ Multiple languages

😲 Reactions and replies

🔒 Encryption

🔎 Search in conversations

✏️ Profile customization

🟢 Status indicators

🎨 Custom themes

📱 Intuitive UI
